Problem
The atmospheric data that explains climate events is scattered across agencies in incompatible formats. Cloud LLMs could reason over it, but 24/7 inference is economically infeasible to self-host, and small local models hallucinate on atmospheric physics.
Approach
AERIS normalizes four atmospheric sources into one schema over a 50 km radius of Houston, flags anomalies with three independent detectors (Z-score, STL decomposition, Isolation Forest), and explains causes with a local Llama 3 8B model grounded by a ChromaDB RAG pipeline. A label-free verification method scores claims using agreement among independent physical sensors. Mentored by Dr. Annalisa Bracco (Georgia Tech / CMCC).

Problem
Analytics dashboards show static, black-box numbers. Nova Core makes a dynamic system where users generate realistic data and trace every insight back to a computed agent output, proving sophisticated ML can stay fully explainable.
Approach
Custom agent orchestration (no LangChain) runs eight agents in dependency order: feature engineering → segmentation → sentiment → churn → recommendations → narrative → audit → query. Churn uses GradientBoosting with per-customer SHAP attribution ("why this customer"), segments use explainable RFM rules, and 45 cross-agent checks validate every run. Each workspace is isolated in its own SQLite database.

Problem
30–40% of household food is wasted, roughly $1,500 a year per family and a major, avoidable source of greenhouse-gas emissions, largely because people forget what's already in their pantry.
Approach
A real-time freshness dashboard with expiration alerts, dual-fallback barcode scanning (native API + ZXing) wired to Open Food Facts for instant product autofill, an "Use It Up" engine that matches recipes to ingredients nearing expiry, and per-household waste analytics that quantify cost and impact. Multi-tenant with Google OAuth, deployed live.

Problem
Screen-time dashboards are abstract and easy to ignore. Detox makes digital wellness tangible by converting tracked time into currency for building and upgrading an island world.
Approach
A local menu-bar agent polls the frontmost app every two seconds, stores sessions in SQLite, and enforces blocks on-device. An optional self-hosted cloud layer syncs over RLS-scoped Postgres with etag-cached rule polling. Privacy-first "Ghost Mode" SHA-256-hashes app names before they ever leave the device. The dashboard is an isometric SVG island with eight themed buildings and a closed reward economy.
